ess.external.powgen.beamline.map_detector_to_spectrum

ess.external.powgen.beamline.map_detector_to_spectrum#

ess.external.powgen.beamline.map_detector_to_spectrum(data, *, detector_info)#

Transform ‘detector’ coords to ‘spectrum’.

Parameters

  • data (DataArray) – Input data whose ‘detector’ coord is transformed.

  • detector_info (DataArray) – Defines mapping from detector numbers to spectra.

Returns

DataArray – data with ‘detector’ coord and dim replaced by ‘spectrum’.
